3 id/name/class区别

“知己知彼，百战不殆”

他们的区别：

  • id：标签唯一标识，好比我们身份证号码，具有唯一性。JS常用document,getGlementBy(id).
  • class：标签的类别，可重复使用，CSS常用。
  • name：标签的名称，与from表单配合使用提交数据后台

id，class一般是页面使用，name一般是后台交互使用
