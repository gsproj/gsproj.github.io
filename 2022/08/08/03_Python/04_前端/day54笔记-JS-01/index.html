1 JavaScript简介

JavaScript(简称JS) 听起来是不是感觉跟Java的关系很大?

其实跟Java什么关系都没有, 只是为了蹭Java的热度,才起的这么个名字!

JS到底是什么?

  • JavaScript 是脚本语言
  • JavaScript 是一种轻量级的编程语言。
  • JavaScript 是可插入 HTML 页面的编程代码。
  • JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。
  • JavaScript 很容易学习

    • True / False
  • 但是在js中布尔值是全小写的
    • true / false
  • 布尔值是false的有哪些
    • 空字符串
    • 0

    • undefined
