find

查找文件，根据文件名、文件大小、文件创建时间，修改时间，访问时间（stat）、文件的权限、文件所属组、文件所属用户、文件类型（普通文件，目录，链接文件（快捷方式））。同时还可以对查询结果做进一步处理。

find的格式

命令     搜索路径              参数            查找条件    指令
find     你需要在哪里去查询文件  按照那种类型搜索  条件表达式   对查询结果的进一步处理

find的参数

find的各个查询条件。

    find   搜索路径  -name   名称表达式   指令
find  /root  -name       "abc"     -print

-size：按照文件大小查询

按照文件的实际大小筛选查询，支持b、c、k、M、G等，默认是b(文件显示的默认单位是：c)

格式
find /root/test -size 大小数值(单位，默认：b)  指令
参数
+n : 表示n到所有

[root@localhost log]# find ./ -size +1k 
./
./tallylog
./lastlog
./wtmp
./tuned/tuned.log
./audit/audit.log
./anaconda/anaconda.log
./anaconda/syslog
./anaconda/X.log
./anaconda/program.log
./anaconda/packaging.log
./anaconda/storage.log
./anaconda/ifcfg.log
./anaconda/journal.log


-n ：表示0到n
[root@localhost log]# find /var/log/ -size -1k
/var/log/btmp
/var/log/anaconda/ks-script-NeF0lj.log
/var/log/boot.log
/var/log/maillog
/var/log/spooler-20210310
/var/log/spooler
[root@localhost log]# find /var/log/ -size -1k
/var/log/btmp
/var/log/anaconda/ks-script-NeF0lj.log
/var/log/boot.log
/var/log/maillog
/var/log/spooler-20210310
/var/log/spooler


n : 表示n个单位
[root@localhost log]# find /root/ -size 489c
[root@localhost log]# find /var/log/  -size 489c
/var/log/grubby_prune_debug
[root@localhost log]# find /var/log/  -size 500c

注： 
	1、n必须为整数
	2、n存在不精确的情况

时间查询

按照不同的时间类型查询。ctime、mtime、atime

格式
find /root  -ctime  时间表达式  指令

n : 代表着天数（整数）
+n : 距离现在n天之前
-n : 距离现在n天之间
n  : 距离现在n天

[root@localhost log]# find ./ -ctime +1
./tallylog
./anaconda/X.log
